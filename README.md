🐹 Go Learning & Challenges
📄 Sobre este Repositório
Este repositório centraliza uma série de desafios técnicos e exercícios práticos que desenvolvi para consolidar meu domínio na linguagem Go (Golang). O objetivo principal é documentar minha evolução na sintaxe da linguagem, no entendimento de concorrência e na aplicação de algoritmos eficientes.

🎯 Foco do Aprendizado
Através destes desafios, explorei conceitos fundamentais e avançados de Go, tais como:

Fundamentos: Variáveis, tipos de dados, structs e interfaces.

Estruturas de Dados: Manipulação de Slices, Maps e Arrays.

Concorrência: Implementação de Goroutines e Channels para processamento paralelo.

Tratamento de Erros: Padrões idiomáticos de Go para construção de aplicações resilientes.

Performance: Escrita de código otimizado e testes unitários.

📂 Estrutura de Pastas
Cada diretório neste repositório representa um desafio ou categoria de exercício, contendo:

Código-fonte (.go): A implementação da solução.

Documentação: Uma breve explicação do problema resolvido (quando aplicável).

Testes (_test.go): Validação da lógica aplicada.

🚀 Por que Go?
Escolhi aprofundar meus conhecimentos em Go devido à sua eficiência em sistemas de back-end de alta performance, tipagem forte e sua abordagem simplificada para a construção de serviços escaláveis, o que complementa meu stack atual em PHP e Python.
